# Ingest Adapters

Data source adapters for reading into Google Sheet Raw Data tabs. Adapter is specified per pipeline in `domain-config.md § Ingest Config`.

| Adapter | File | Use When |
|---------|------|----------|
| sheets | `sheets.md` | Source is a Google Sheet tab |
| csv | `csv.md` | Source is a local CSV/Excel or Drive file |

See each adapter file for input/output contract and error handling.
